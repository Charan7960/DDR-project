How It Works

Our ML-powered content-based filtering pipeline

01

Data Collection

TMDB 5000 dataset with movies, genres, cast, crew, keywords, and user ratings from Kaggle.

02

Feature Engineering

Extract & combine genres, keywords, top 5 cast, director into a unified feature "soup" per movie.

03

TF-IDF Vectorization

Transform text features into numerical TF-IDF vectors (15,000-dim) using scikit-learn.

04

Cosine Similarity

Compute pairwise cosine similarity matrix to measure how alike each movie pair is.

05

Ranked Results

Sort by similarity score & return top-N recommendations with match percentage.
